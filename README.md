# 대회 / 코딩테스트 알고리즘 정리
알고리즘의 **동작 원리와 구현 방법**을 중심으로 정리한 노트입니다.

대회와 코딩 테스트에서 자주 쓰이는 자료구조와 알고리즘을, 문제를 풀 때 바로 꺼내 볼 수 있도록 **간단하고 직관적으로** 정리했습니다.

이 저장소는 엄밀한 이론서보다는, **실전 문제 풀이를 위한 빠른 참고서**에 가깝게 구성되어 있습니다.

## 정리 방식
이 문서는 다음 기준으로 작성되었습니다.

- 설명은 가능한 한 짧고 직관적으로 정리
- **동작 이해 → 구현 확인 → 연습 문제 적용** 흐름으로 구성
- 엄밀한 증명이나 깊은 이론은 최소한만 다룸
- 실전에서 자주 쓰는 형태의 구현을 우선 소개

## 대상 독자
이 문서는 다음과 같은 사람을 대상으로 합니다.

- 자료구조와 알고리즘을 빠르게 훑고 싶은 사람
- 코딩 테스트를 준비하는 사람
- 백준 문제를 꾸준히 푸는 사람
- 알고리즘 문제 풀이 실력을 기르고 싶은 사람
- 알고리즘 구현 템플릿을 정리해 두고 싶은 사람

## 난이도 표기 기준
각 알고리즘 옆의 난이도는 작성자의 경험을 바탕으로 한 **참고용 기준**입니다.

- 절대적인 난이도를 의미하지 않습니다.
- 해당 알고리즘이 본격적으로 등장하기 시작하는 문제 난이도의 **대략적인 하한선** 정도로 보면 됩니다.
- 실제 문제에서는 더 높은 난이도와 추가 응용이 필요할 수 있습니다.

또한 **목차의 순서는 정확한 난이도 순서를 의미하지 않습니다.**  
학습 흐름이 비교적 자연스럽도록 배치한 순서입니다.
