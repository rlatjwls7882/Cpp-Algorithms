# 대회 / 코딩테스트 알고리즘 정리

정확한 증명이나 자세한 이론적 배경보다는, 대회와 코딩 테스트에서 바로 활용할 수 있는 이해와 구현에 초점을 맞추었습니다.

## 난이도 표기 기준
각 알고리즘 옆의 난이도 표기는 작성자의 경험을 바탕으로 한 참고용 기준입니다.  
정확한 난이도를 의미하지 않으며, 해당 알고리즘이 등장하는 문제의 **최소 티어 정도**로 생각하면 됩니다.

또한 차례의 순서는 정확한 난이도 순서를 의미하지 않습니다.
