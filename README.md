# 알고리즘 정리
대회와 코딩 테스트에서 자주 사용하는 자료구조와 알고리즘을 **동작 원리, 구현 방법, 활용 방식** 중심으로 정리한 문서입니다.

이 저장소는 엄밀한 이론서보다는, 문제를 풀다가 필요한 내용을 빠르게 다시 확인할 수 있는 **실전형 참고서**를 목표로 합니다.

## 문서 구성
이 문서는 다음 기준으로 작성되었습니다.

- 핵심 개념을 가능한 한 짧고 직관적으로 설명
- **동작 이해 → 구현 확인 → 연습 문제 적용** 흐름으로 구성
- 엄밀한 증명이나 세부 이론은 필요한 범위에서만 다룸
- 실전에서 바로 활용할 수 있는 구현 형태를 우선 정리
- 학습 흐름을 고려해 목차를 구성하되, 정확한 난이도순을 의미하지 않음

## 대상 독자
다음과 같은 독자를 대상으로 합니다.

- 자료구조와 알고리즘을 빠르게 훑고 싶은 사람
- 코딩 테스트를 준비하는 사람
- 백준 등 알고리즘 문제 풀이를 꾸준히 하는 사람
- 알고리즘 구현 템플릿과 핵심 아이디어를 함께 정리하고 싶은 사람

## 난이도 표기 기준
문서에 표시된 난이도는 **참고용 기준**입니다.

- 절대적인 난이도를 의미하지 않습니다.
- 각 알고리즘이 본격적으로 등장하기 시작하는 문제 난이도의 **대략적인 하한선**에 가깝습니다.
- 실제 문제에서는 더 높은 난이도와 추가 응용이 필요할 수 있습니다.
