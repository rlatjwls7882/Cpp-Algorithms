# 대회 / 코딩테스트 알고리즘 정리

알고리즘의 **동작 원리와 구현 중심**으로 정리한 알고리즘 노트입니다.  
대회와 코딩 테스트에서 **바로 활용할 수 있는 이해와 구현**을 목표로 작성했습니다.

이 문서는 **백준 / Codeforces / AtCoder / ICPC 스타일 문제 풀이**를 공부하는 사람을 대상으로 합니다.

## 난이도 표기 기준
각 알고리즘 옆의 난이도는 작성자의 경험을 바탕으로 한 **참고용 기준**입니다.

절대적인 난이도를 의미하지 않으며 해당 알고리즘이 등장하는 문제의 **최소 티어 정도**로 생각하면 됩니다.

또한 목차의 순서는 **정확한 난이도 순서를 의미하지 않습니다.**
